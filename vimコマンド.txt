◆便利
""を消去してInsert Modeに移る：ci"
単語を消去してInsert Modeに移る：ciw
右の{char}の真上に移動する：f{char}
右の{char}の直前の文字(character)移動する：t {char}

◆繰り返し
直前のコマンドを再実行：.(ドット)
n行分削除:ndd
n行分上へ：nk
n個分順方向の文字の上に移動：nf{char}
;:順方向に繰り返し検索
::逆方向に繰り返し検索
abcdefg abcdefg abcdefg
aから三つ目のeに行く場合→→fe ; ;

◆選択系とコピー、ペースト系
カーソル行の末尾に改行：o
カーソル行の先頭に改行：O
コピーしたい場合:y
カットしたい場合:d
行単位選択：Shift + v
カーソル位置の行をヤンク:yy
カーソル位置の行からn行目までをヤンク：ynG
最初の行をヤンク：ygg
最後の行をヤンク：yG
指定の文字数分ヤンク：{number}yl
単語のヤンク：yiw
カーソル位置の後ろにペースト：p
直前の操作を取り消し：u
カーソル位置の行を削除：dd
カーソル位置の行を削除してインサートモード：CC
カーソル位置の行からn行目までをヤンク：dnG
Delete:dl
指定の文字数分削除：{number}dl
カーソル位置から行末まで削除：D
カーソル位置から行末まで削除してインサートモード：C
Back space:dh

短形選択モードでインデントを上げ下げする：
１）Shift+v
２）行選択（J or K）
３）上げる場合は>
　　下げる場合は<

◆移動系
行頭へ：0
行末へ：$
次の単語へ：w
前の単語へ：b
左へ：h
右へ：l
上へ：k
下へ：j
指定の行(53行目）へ移動：53G
ファイルの先頭に移動：gg
単語の終端に移動：e
ファイルの末尾に移動：G
f:その行の順方向に一文字検索
F:その行の逆方向に一文字検索
t:その行の順方向に一文字検索(カーソルは文字の手前)
T:その行の逆方向に一文字検索(カーソルは文字の手前)
ctrl+u:半画面分 上へ
ctrl+d:半画面分 下へ
zt:先頭行に合わせる
zb:最下行に合わせる

◆検索・置換
前方検索：/pattrn
後方検索：?pattrn
検索結果のハイライトを消去する：:noh
カーソル位置の単語を検索して移動する：*n
指定の行の指定の文字ないし文字列を置換する：<開始行番号>,<終了行番号>s/<置換前>/置換後/g
例）122行目のTest2をTest3に置換したい場合
:122s/2/3/g

ファイル全体で確認をしながら置換
:%s/置換前/置換後/gc
例）以下の選択ができる
y:置換する
n:置換しない
q(Esc):置換モード終了
a:現在のマッチとそれ以降の結果を全て置換する

◆タブ操作
タブの新規作成：:tabnew
次のタブに移動：gt
複数のファイルをタブで開く：vim -p file1.txt file2.txt 
現在のタブを閉じる：:tabclose
タブを破棄して閉じる：:tabclose!

◆ターミナル
ターミナルを開く：:terminal

◆分割された画面
カーソルを1つ下のwindowに移動：<Ctrl+w>j
カーソルを1つ上のwindowに移動：<Ctrl+w>k
現在のwindowを画面の一番下に移動する：<Ctrl+w>J
現在のwindowを画面の一番上に移動する：<Ctrl+w>K
現在のwindowの縦幅を広げる：<Ctrl+w>+

◆レジスタ
レジスタの内容を表示：:registers
レジスタからテキストをペースト："レジスタ名p
指定のレジスタにヤンク："レジスタ名y

◆ファイル操作
直前の操作を取り消し(アンドゥ)：u
戻る（リドゥ）:ctrl+r
保存：:w
終了：:q
保存して終了：:wq

◆モード切り替え
ビジュアルモード：v
インサート位置からモード：i
行末に移動しインサートモード：A
行頭に移動しインサートモード：I
ノーマルモード：ctrl + [

◆vimrcでの設定項目
set number
set hlsearch
set cursorline
set guioptions-=m
syntax on
set autoindent
colorscheme blue
" 挿入モードでバックスペースで削除できるようにする
set backspace=indent,eol,start
" 検索結果がファイル末尾まで進んだら、ファイルっ先頭から再び検索
set wrapscan
" 挿入モード中にカーソルを移動するマッピング
inoremap <C-h> <Left>
inoremap <C-j> <Down>
inoremap <C-k> <Up>
inoremap <C-l> <Right>
" バックアップファイルを作成しない
set nobackup
set noswapfile
" カーソルを行末の一時文超えるようにせてい
set virtualedit=onemore
" ビープ音を消す
set belloff=all
" マッチ件数を表示
set shortmess-=S
" クリップモードとヤンク
set clipboard+=unnamedplus,unnamed
" フォントとサイズ
set guifont=Consolas:h20
